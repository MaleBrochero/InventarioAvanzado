# Advanced Inventory

## Technology used: Python.

_This is a program with multiple interactive tools, providing advanced features since you can adjust and use different functions.

This project manages a company's inventory. The inventory is dynamic and includes the following functions: add, display, search, update, delete, statistics, save CSV, load CSV, and exit (which means ending the program execution).

The program runs in the console with a numeric menu. Follow the instructions as indicated by the program._
